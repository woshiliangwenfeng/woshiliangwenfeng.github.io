## How to Add Missing Assets

### Favicon
1. Create a 32x32 or 16x16 pixel image
2. Convert to .ico format using online tools:
   - https://www.favicon-generator.org/
   - https://realfavicongenerator.net/
3. Save as `favicon.ico` in the root directory

### Open Graph Image
1. Create a 1200x630 pixel image
2. Include your branding and/or game-related content
3. Use clear, readable text if including any
4. Save as `og-image.jpg` or `og-image.png` in the root directory
